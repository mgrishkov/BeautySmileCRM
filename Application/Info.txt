Порядок установки:
Пункты 1-6 выполняются только на сервере (ПК, где будет установлена СУБД), на других ПК установка выполняется с п.7.

Установка и настройка MS SQL Server'а
1.	Установить MS SQLServer версии 2012 (для Windows 7, Windows Server 2008 и выше) или версии 2008R2 для более ранних врсий Windows.
	Ссылка для скачивания SQLSever 2012 http://www.microsoft.com/ru-ru/download/details.aspx?id=29062. 
	В открывшемся окне выбрать пункт:
	- 	RUS\x64\SQLEXPR_x64_RUS.exe для 64х разрядных систем
	- 	RUS\x86\SQLEXPR32_x86_RUS.exe для 32х разрядных систем
	По зервешению загрузки апустить установку. В открывшемся окне выбора инстанса (Instance) указать По-умолчанию (Default Instance).
	В окне настроек параметров авторизации, указать смешанный режим (Mixed mode - SQLServer и Windows) и указать пароль для системной учетной записи.
	Продолжить установку со стандартными параметрами.
	
2.	Для сетевого доступа необходимо выполнить настройку SQLServer'а.
	Переходим в "Панель управления" -> "Администрирование" -> "Управление компьютером". В дереве консоли раскрываем раздел "Службы и приложения" -> 
	"SQL Server Configuration Manager" -> "SQL Server Network Cinfiguration" -> "Protocols for MSSQLSERVER". В таблице справа активируем все три пункта:
	"Shared Memory", "Named Pipes", "TCP/IP". Для этого щелкаем по ним правой кнопкой мыши и выбираем "Enable".
	
3.	Перезагружаем ПК.
	
4.	Проверка установки и настроек SQL Server'а.
	Нажимаем кнопки Win+R. Пишем комманду cmd и в открывшемся окне пишем sqlcmd и нажимаем Enter.
	На новой строке после знака > пишем select 1 нажимаем Enter и на след строке после знака > пишем go
	В результате должна появится надпись: (1 row affected)
		
Установка приложения:
7.	Из корневой папки запускаем setup.exe. Во время установки будет произведена проверка на необходимые и компоненты и, если они отсутствуют, появится 
	запрос на их скачивание и установку. В таком случае соглашаемся и продолжаем установку. По завершению установки доп. компонентов система выдаст запрос
	на перезагрузку. Перезагружаемся. После загрузки, установка будет продолжена. 
	По окончанию установки в меню пуск и на рабочем столе появятся ярлыки BeautySmileCRM.
	
Первоначальный запуск:
8.	Для первого запуска на сервере в окне авторизации необходимо указать имя пользователя sysdba, сервер localhost и пароль qwerty~123. После чего приложение
	само создаст для себя Базу данных и обновит ее до нудной версии. Пользователь sysdba
	- системный, имеет все привилегии и может быть использован только с ПК, на котором установлен сервер БД. 

Обновление:
	В случае предоствления обновления приложения, необходимо обновить его по аналогии с установкой (п.7). После обновления необходимо выполнить шаги из п.8.

Удаление БД без возможности восстановления:
	Для полного удаления БД, запустите файл Service\DropDB.bat

Создание копии БД:
	Для создания резервной копии БД, запустите файл Service\BackupDB.bat. По завершению работы файл резервной копии будет сохранен в папку C:\Temp\CRM.bak

Восстановление резервной копии:
	Для восстановления резервной копии БД, запустите файл Service\RestoreDB.bat. Файл резервной копии долежен быть расположен в C:\Temp\CRM.bak
